
 # Electricity Bill Calculation System

## Project Overview
This is a Java console application that calculates the monthly electricity bill for a consumer. The bill is calculated based on the number of units consumed using a multi-tier slab rate system.

## Key Features
1. Stores consumer details like ID, Name, and Units Consumed.
2. Calculates the total bill using 4 different price slabs.
3. Adds a fixed charge of ₹125 to the final bill amount.
4. Uses separate classes for data and logic for clean code design.
5. Checks for invalid inputs like negative units.

## How to Run the Program
1. Make sure you have Java JDK installed on your system.
2. Save the code in a file named `ElectricityBillSystem.java`.
3. Open Command Prompt or Terminal in the same folder.
4. Compile the code using the command:javac_.java
5. Run the program using the command:java ElectricityBillSystem
